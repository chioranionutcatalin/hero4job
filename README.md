# Hero4Job

Hero4Job is a CV builder app where users fill in personal, experience, skills, language, and education forms, then preview and download their CV as a PDF.

The app includes:
- Interactive form-based CV editing
- Two visual CV layouts (Classic and Compact)
- Home page with model CV templates
- Preview page for submitted user data
- PDF export of the selected layout

## Run locally

```bash
npm install
npm run dev
```
